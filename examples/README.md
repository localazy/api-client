# 📦 `@localazy/api-client` examples

Node 16+ is required.

> Run all commands from project root.

## 🔧 Install

```bash
npm install
```

## 🔐 ENV variables

Project token and api url is passed by ENV variables. Create `.env` file from example and update your project token.

```bash
cp .env.example .env
```
